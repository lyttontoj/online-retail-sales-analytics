# Online Retail Sales Analytics using Tableau

## Project Overview

This project analyzes online retail transaction data to identify sales trends, customer purchasing behavior, product performance, and geographical revenue distribution.

The project was developed as part of the MSc Big Data Management and Analytics program at Griffith College Dublin.

## Objectives

- Analyze sales and revenue trends
- Perform customer segmentation
- Evaluate product performance
- Study geographical revenue contribution
- Generate predictive business insights

## Tools Used

- Tableau
- Excel
- Data Analytics
- Exploratory Data Analysis (EDA)
- Business Intelligence

## Dataset

The dataset used in this project is the Online Retail Dataset from Kaggle:

https://www.kaggle.com/datasets/carrie1/ecommerce-data

Due to GitHub file size limitations, the original dataset is not included in this repository.

## Key Insights

### Revenue Analysis
- Identified seasonal sales trends
- Analyzed monthly revenue growth

### Customer Analytics
- Segmented repeat and one-time customers
- Studied purchase frequency patterns

### Product Performance
- Identified top revenue-generating products
- Analyzed quantity vs price relationship

### Geographical Analysis
- Compared revenue across countries
- Identified high-performing markets


## Business Value

The dashboards help organizations:

- Improve customer retention
- Optimize inventory planning
- Enhance pricing strategies
- Support regional expansion decisions
